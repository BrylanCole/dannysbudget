Excel Sync

Power Query setup

How it works

This site publishes your budget data as a JSON file at a public URL on GitHub Pages. Excel's Power Query can read that URL and refresh automatically on a schedule — so your spreadsheet always reflects what's on your site.

Step 1 — Deploy to GitHub Pages
1

Create a GitHub repository

Go to github.com → New repository. Name it budget and set it to Public.

2

Upload your files

Upload index.html and data.json (download below) to the repo root.

3

Enable GitHub Pages

Settings → Pages → Source: Deploy from branch → main / root. Your site goes live at https://yourusername.github.io/budget/

4

Your data URL

Your Power Query URL will be:

https://yourusername.github.io/budget/data.json
Step 2 — Connect Excel via Power Query
1

Open Power Query

In Excel: Data tab → Get Data → From Other Sources → From Web

2

Paste your data URL

Enter your GitHub Pages data URL and click OK. Power Query will load the JSON.

3

Transform and load

In Power Query Editor, click the transactions list → Convert to Table → expand columns. Click Close & Load.

4

Set auto-refresh

Data → Queries & Connections → right-click your query → Properties → check "Refresh every X minutes" and "Refresh data when opening the file".
